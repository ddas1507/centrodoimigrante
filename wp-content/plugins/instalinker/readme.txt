**How to Add Instagram to WordPress – Automatic Installation (Recommended)**

1. Go to `Plugins > Add New`
2. Search for "InstaLink Lite" and install it
3. Activate your version

**How to Embed Instagram on WordPress – Manual Installation**

1. Download `instalinker.1.4.1.zip` from `wordpress.org` and unzip it
2. Upload the `instalinker` folder to `/wp-content/plugins/`
3. Go to `Plugins > Installed Plugins` and activate the newest widget

= InstaLink Shortcode Example =

This is what you will get exactly after making customization of your photo gallery. The shortcode includes information about all options and their values. You will be able to add Instagram gallery to WordPress with its help.

`[instalink username="elfsight" width="350px" height="350px" image_size="small" bg_color="#285989"]`

= Available Instagram Gallery Options =

* **width** - grid’s width (any CSS valid value: px, %, em, etc.). Set it as “auto” to get Instagram widget that fits automatically screen sizes.
* **height** - grid’s height (any CSS valid value: px, %, em, etc.)
* **image_size** - display Instagram photos with “small” or “xlarge” size
* **bg_color** - header and follow button background color (any CSS valid value)

Questions You Might Ask Concerning Our Instagram Widget InstaLink Lite

= How can I use InstaLink Lite shortcode? =
You can add shortcode of our Instagram widget to any place of your website. There are 3 variants of using Instagram plugin's shortcode:

1. You can insert it into content of any web page or blog post. Just copy the shortcode of your Instagram widget to the Editor page when you edit a page or a post.

2. You can insert the Instagram widget shortcode into text widget. It might be useful if it’s required to add the Instagram gallery to a sidebar. Note that your theme should include a special filter: `add_filter( 'widget_text', 'do_shortcode' );` If your theme doesn’t include it then it’s necessary to add this filter to functions.php of the theme. After that drag and drop text widget into any sidebar and copy the shortcode of Instagram gallery into text widget.

3. You can insert the Instagram widget shortcode into a template, footer or any other php file of your WordPress theme. To do this use the following function: `<?php do_shortcode('[instalink]'); ?>`. Learn more about this function on [https://developer.wordpress.org/reference/functions/do_shortcode/](https://developer.wordpress.org/reference/functions/do_shortcode/)
